# RR WhatsApp API - Auto Instalador By Bento

Este repositório contém um script de instalação automática para configurar a **RR WhatsApp API** em qualquer máquina Linux com apenas uma linha de comando.

---

## Pré-requisitos

Antes de executar o instalador, certifique-se de ter:

- Linux (Ubuntu, Debian ou derivados)
- `curl` ou `wget` instalado
- `bash` disponível
- Acesso root ou sudo

---
